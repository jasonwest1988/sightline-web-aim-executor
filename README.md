## What It Is

SIGHTLINE is a browser-first aim trainer made for desktop use. It delivers practice modes for flicking, tracking, reaction drills, and quick target transitions, all without needing a separate installation.

The tool is built around short repeatable practice blocks and longer warmup sessions alike. It also keeps local score and accuracy stats, while giving you controls for drill tuning and crosshair presentation.

---

## Features

- Works completely in the web browser, so no install step is needed
- Flick-focused drills for snapping onto targets quickly
- Tracking exercises for staying on moving targets
- Reaction scenarios for timing and response practice
- Multi-target switching to improve transition speed
- Controls for target size, spawn speed, and session duration
- Crosshair style options for personal preference
- Score and accuracy data saved locally in the browser

## Configuration

The training interface exposes the main session controls directly.

| Setting | Purpose |
| --- | --- |
| Target size | Changes how large the targets appear |
| Spawn speed | Controls how quickly targets appear or move |
| Session length | Sets how long each training run lasts |
| Drill type | Switches between flick, tracking, reaction, and switch modes |
| Crosshair style | Changes the look of the on-screen crosshair |
| Local stats | Keeps score and accuracy history in browser storage |

Example configuration ideas:

- Short warm-up: small targets, faster spawn speed, short session
- Precision practice: medium targets, slower pace, longer session
- Reaction work: quick target appearance with switch drills enabled

---

## Browser Support

SIGHTLINE is intended for modern desktop browsers and is built to run directly in-browser with no extra dependencies. It uses browser storage for local score and accuracy tracking, so clearing site data can remove saved session history.

Known limitations:

- Best suited for desktop use rather than mobile browsers
- Performance can vary by browser and system resources
- Local data remains tied to the browser profile being used

## FAQ

### How do I start using it?
Open the project in a supported desktop browser and load the main page. If you are working from local files, make sure the HTML file and its assets stay together.

### Can I update it easily?
Yes. Swap in the newer files and reopen the trainer in your browser.

### Can I customize the drills?
Yes. Target size, spawn speed, session length, drill type, and crosshair appearance can all be adjusted.

### Does it work everywhere?
It is designed for modern desktop browsers. Older browsers or restricted environments may not support all features.

### Where is my progress saved?
Score and accuracy data are stored locally in the browser. If you clear browser storage, that data can be removed.
